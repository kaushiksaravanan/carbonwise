# CarbonWise — 90-Second Demo Script

## Setup
- Tools needed: Browser, screen recorder (Loom/OBS)
- Prepare: Open https://carbonwise-mu.vercel.app in incognito mode (fresh state)
- Voice: Friendly, energetic, conversational

## Scene 1: Hook (0:00 - 0:08)
**On screen**: Landing page with tree-grow Lottie animation
**Voiceover**: "What if reducing your carbon footprint felt as fun as growing a video game garden?"
**Action**: Linger on the tree animation
**Visual notes**: Hold steady on the Lottie animation; let the tree fully grow before cutting. Soft zoom-in (5%) for subtle motion.

## Scene 2: The Problem (0:08 - 0:18)
**On screen**: Quick cuts showing typical carbon calculator UIs (boring forms)
**Voiceover**: "Most carbon apps are abandoned within a week. They're calculators, not coaches. Numbers, not motivation."
**Action**: Cut between 3-4 stock screenshots of generic calculator UIs (1-2 seconds each)
**Visual notes**: Use desaturated/grayscale filter on these clips to contrast the colorful CarbonWise reveal coming next. B-roll suggestions: any spreadsheet-style carbon calculator screenshot.

## Scene 3: Onboarding (0:18 - 0:35)
**On screen**: Click "Get Started" — quickly answer all 5 questions
**Voiceover**: "Meet CarbonWise. In 30 seconds, it learns your lifestyle..."
**Action**: Fast-forward through quiz (2x speed), show progress bar filling
**Visual notes**: Speed up cursor movement in post; keep the progress bar animation at normal speed for impact. Add subtle "whoosh" SFX between questions.

## Scene 4: Dashboard Reveal (0:35 - 0:50)
**On screen**: Dashboard appears with budget ring, garden, streak
**Voiceover**: "...and gives you a daily budget. A living garden. And an AI coach who actually knows YOU."
**Action**: Hover over the carbon budget ring, show the garden
**Visual notes**: This is the "wow" moment. Use a slight bloom/glow effect on first reveal. Pan slowly across the three key UI elements (budget → garden → coach) as each is mentioned.

## Scene 5: AI Coach Magic (0:50 - 1:05)
**On screen**: Click chat, type "What's my biggest opportunity to reduce CO2?"
**Voiceover**: "Powered by Gemini, the coach responds with advice tailored to YOUR habits — not generic platitudes."
**Action**: Show the AI's personalized response
**Visual notes**: Type at realistic speed (don't fast-forward — the typing builds anticipation). When response streams in, zoom in 10% on the chat bubble to highlight personalization.

## Scene 6: What-If Simulator (1:05 - 1:18)
**On screen**: Click simulator, click "I bike to work 3 days/week"
**Voiceover**: "Curious what change has the biggest impact? Simulate it before committing."
**Action**: Show the bar chart with savings
**Visual notes**: Capture the chart animation in real-time. Add a subtle highlight ring around the savings number when it appears. This sells the "evaluation" angle for judges.

## Scene 7: The Garden Grows (1:18 - 1:25)
**On screen**: Back to dashboard, garden has more flowers/trees
**Voiceover**: "Stay under budget — your garden grows. Make low-carbon choices a habit, not a chore."
**Action**: Show before/after of the garden state with more flora
**Visual notes**: Quick split-screen or wipe transition between empty garden and grown garden. Adds satisfying payoff.

## Scene 8: Call to Action (1:25 - 1:30)
**On screen**: URL prominent: carbonwise-mu.vercel.app
**Voiceover**: "Try CarbonWise free at carbonwise-mu.vercel.app. No signup. No tracking. Just impact."
**Visual notes**: Full-screen URL with logo. Hold for 3+ seconds. Music swells then fades.

## Tips for Recording
- Use 1920x1080 resolution
- Record in 60fps for smoother UI animations
- Background music: light, optimistic, royalty-free (Bensound's "Sunny" works)
- Captions: add for accessibility
- End card: 3 seconds with logo and URL

## Filming Checklist

### Pre-Recording
- [ ] Close all browser tabs except CarbonWise
- [ ] Hide bookmarks bar (Ctrl+Shift+B)
- [ ] Open in incognito/private window for fresh onboarding state
- [ ] Clear localStorage if you've used the app before (DevTools → Application → Clear)
- [ ] Set browser zoom to 100% (Ctrl+0)
- [ ] Disable browser notifications and Do Not Disturb mode
- [ ] Close Slack, email, and other notification apps
- [ ] Plug in laptop (no battery saver dimming)
- [ ] Test microphone levels — record 10 seconds and play back
- [ ] Run app once to confirm Gemini API responds (don't want a 500 mid-take)
- [ ] Pre-write the chat prompt so you don't fumble typing on camera

### Equipment & Software
- [ ] Screen recorder configured for 1920x1080 @ 60fps
- [ ] OBS scene set up (or Loom Pro for built-in editing)
- [ ] External mic or quality headset (laptop mic = last resort)
- [ ] Quiet room — close windows, turn off fans/AC if loud
- [ ] Backup: have a phone recording audio as a safety track

### During Recording
- [ ] Record voiceover separately from screen capture for cleaner audio
- [ ] Take multiple takes of each scene — costs nothing in editing
- [ ] Move cursor deliberately and slowly; viewers' eyes follow it
- [ ] Pause 1 second before each click (gives editing room)
- [ ] Speak with energy — flat delivery kills demo videos

### Post-Production
- [ ] Trim dead air at start/end of each scene
- [ ] Add background music at -18dB (under voiceover)
- [ ] Duck music during voiceover (auto-ducking in most editors)
- [ ] Add captions/subtitles (CapCut, Descript, or YouTube auto-caption)
- [ ] Color-grade the "boring competitor" clips desaturated
- [ ] Add zoom/pan (Ken Burns) on static frames to keep energy up
- [ ] Export at 1080p, H.264, ~10 Mbps bitrate
- [ ] Create vertical 9:16 cut for TikTok/Reels/Shorts
- [ ] Create square 1:1 cut for Twitter/LinkedIn

### Distribution
- [ ] Upload master 16:9 to YouTube (unlisted first for review)
- [ ] Get feedback from 1-2 trusted reviewers before public posting
- [ ] Write a strong hook caption for each platform
- [ ] Tag with #climatetech #AI #sustainability #hackathon
- [ ] Add link to carbonwise-mu.vercel.app in description AND first comment
- [ ] Pin the demo on your Twitter/LinkedIn profile

### Final Polish
- [ ] Watch the full video on mute — does it tell the story visually?
- [ ] Watch with captions only — does it make sense without sound?
- [ ] Watch on phone — is text readable at small size?
- [ ] Time it precisely — under 90 seconds beats exactly 90
